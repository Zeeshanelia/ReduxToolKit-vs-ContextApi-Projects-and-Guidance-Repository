# E-Commerce Cart Redux TK LocalStorage

A modern React + Redux Toolkit shopping cart application with persistent cart using localStorage.

## Features
- Current Approach (Manual localStorage in cartSlice)
- Browse products from DummyJSON API
- Add / Remove / Update quantity in cart
- Real-time cart total calculation
- Cart persistence using localStorage
- Clean and responsive UI
- manually call localStorage.setItem() in every reducer (addItem, removeItem, decreaseQuantity, clearAllItems).
